快速搭建开箱即用的 `storybook`组件库开发环境。

- 初始化速度快（官方的初始化太慢了）
- 开箱即用（已填坑版）

## 安装

```shell
npm i -g @miapub/storybook-cli
```

## 初始化项目

```shell
sb-cli create <项目名> <Path>

# 若想初始化内容就放在当前根目录，path参数可使用 .
sb-cli create xx .
```
